# Feed neighbor

Feed neighbor is a simple app that serves as a platform to share food between neighbors.


## Applied skills

* HTML5
* Javascript ES8
* Vue + Vuex + Vuetify

## Running

First install all dependencies:

```npm install```

Next run the app:

```npm run serve```
